# zh-portal-lab

## 项目简介

zh-portal-lab 是一个用于归档和发布多个独立 HTML 页面的仓库。本仓库不针对任何特定域名或网站，仅作为页面的集中存放与版本管理空间。

## 目录说明

- `pages/`：存放各个独立 HTML 页面文件，每个页面可包含完整的样式与脚本。
- `assets/`：共用的资源文件，如图片、样式表、JavaScript 脚本等。
- `docs/`：与页面相关的说明文档或参考材料。
- `templates/`：可复用的页面模板，便于快速创建新页面。

## 页面归档说明

本仓库中的 HTML 页面均为独立归档，彼此之间无依赖关系。每个页面均可在浏览器中直接打开运行，或部署至任意静态服务器。归档时建议保留清晰的命名与简短备注，便于检索与维护。

## 维护说明

- 新增页面请放置在 `pages/` 目录下，并确保文件名具有辨识度。
- 如需引用共用资源，请使用相对路径指向 `assets/` 目录。
- 提交前请检查页面是否能在无网络环境下正常渲染（除非页面明确依赖外部资源）。
- 鼓励在提交信息中简要描述页面的用途或变更内容。
